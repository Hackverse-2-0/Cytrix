# Echo-Tremor-Insight

## 📌 Project Overview
Echo-Tremor-Insight is a web-based application designed to analyze and visualize tremor-related data.  
The platform provides an interactive dashboard for better understanding and monitoring tremor patterns.

---

## 🚀 Features
- Interactive data visualization
- Clean and responsive UI
- Real-time analysis support
- Modern dashboard design

---

## 🛠️ Tech Stack
- React (Vite)
- Tailwind CSS
- shadcn/ui
- Node.js
